About This
Project

A data science capstone project built for Goldman Sachs Ayco — demonstrating full-stack ML deployment, quantitative finance, and AI-powered financial analysis.

Project Architecture

Built using the same Flask deployment pattern as a production ML system — the financial data model is pickled and loaded at runtime, mirroring how a trained predictive model would be served (same pattern as DTSC-680 Assignment 5).

application.py — Flask backend, three route modules
build_financial_model.py — builds & pickles model
financial_model.pkl — pickled rate/benchmark data
templates/ — Jinja2 HTML frontend
Three Modules
01 — RAG Research Assistant
Retrieval-augmented generation over a 15-entry curated financial knowledge base. Keyword scoring with topic-title boosting. Grounded, cited responses with zero hallucination outside the knowledge base.
02 — Black-Scholes Options Pricer
Pure-Python implementation of the Black-Scholes formula using the Abramowitz & Stegun normal CDF approximation. Computes fair value, intrinsic value, time value, moneyness, and five Greeks (Δ, Γ, Θ, V, σ).
03 — Portfolio Risk Dashboard
Parametric VaR (95% confidence, 30-day) using Z-score of 1.645. Sharpe Ratio calculated against the current 10Y Treasury risk-free rate (4.25%). Volatility classification and risk profile matching across 5 asset classes.
